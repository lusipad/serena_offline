# Contributing to Serena Offline Builder

Thank you for your interest in contributing!

## Getting Started

1.  Clone this repository.
2.  Ensure you have Python 3.11+ and `uv` installed.
3.  Clone the main [Serena repository](https://github.com/lusipad/serena) locally as you will need it to test the builds.

## Reporting Bugs

Please open an issue on GitHub with:
*   Your operating system version.
*   Python version.
*   The specific error message or behavior observed.
*   Logs from the GUI builder console.

## Pull Requests

1.  Fork the repository.
2.  Create a new branch for your feature or fix.
3.  Make your changes.
4.  Test the builder locally to ensure it generates a working `dist/serena-standalone` package.
5.  Submit a Pull Request.

## Coding Style

*   We follow PEP 8 for Python code.
*   Please use `.editorconfig` settings if your editor supports it.
